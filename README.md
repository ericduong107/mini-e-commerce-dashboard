# mini-e-commerce-dashboard
Mini E-commerce Dashboard

## Cấu trúc thư mục dự án

```
src/
├── api/             # Cấu hình Axios, gọi API
├── components/      # Các UI nhỏ (Button, Card, Input) - Tái sử dụng
├── features/        # Các tính năng lớn (ProductList, Cart)
├── store/           # Cấu hình Redux (Slices)
├── layouts/         # Bố cục trang (Header, Footer)
├── hooks/           # Custom hooks (VD: useFetch, useDebounce)
├── utils/           # Các hàm phụ trợ (Format tiền tệ)
└── types/           # Định nghĩa TypeScript interfaces
```
